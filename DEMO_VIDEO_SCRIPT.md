## 🎥 RECORDING TIPS

### Equipment Needed:
- **Screen Recording:** OBS Studio (free), Loom, or Windows Game Bar (Win+G)
- **Microphone:** Any headset mic works
- **Optional:** Webcam for intro/outro

### Recording Steps:

1. **Prepare the app:**
   - Open http://localhost:3000 (or your Vercel URL)
   - Clear any previous data (refresh)
   - Have test messages ready to copy-paste

2. **Screen recording:**
   ```
   Windows: Press Win+G → Click Record
   Or download OBS Studio: https://obsproject.com
   ```

3. **Record in sections:**
   - Record intro separately (with webcam if you want)
   - Record screen demo
   - Record outro
   - Combine in video editor

4. **Free Video Editors:**
   - CapCut (easiest): https://www.capcut.com
   - DaVinci Resolve (professional): https://www.blackmagicdesign.com/products/davinciresolve
   - Clipchamp (built into Windows 11)

### Upload to YouTube:

1. Go to https://studio.youtube.com
2. Click **Create** → **Upload Video**
3. Set visibility to **Unlisted** (only people with link can see)
4. Add title: "SafeHer AI - AI-Powered Women's Safety App | [Hackathon Name]"
5. Add description with features list
6. Copy the link for submission
